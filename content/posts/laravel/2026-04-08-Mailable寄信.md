Laravel 可使用 Mailable 機制建立簡單的電子郵件寄送功能。

## 建立 Mailable

建立名稱為 OrderShipped 的 Mailable 類別

```bash
php artisan make:mail OrderShipped

# 將會在 app/Mail 目錄生成 OrderShipped
```

編輯 app/Mail/OrderShipped.php

```php
class OrderShipped extends Mailable
{
    use Queueable, SerializesModels;

    // 定義公有屬性，Blade 視圖會自動取得此變數
    public $order;

    public function __construct(Order $order)
    {
        $this->order = $order;
    }

    // 設定郵件主旨與寄件人（選填）
    public function envelope(): Envelope
    {
        return new Envelope(
            subject: '您的訂單已出貨：' . $this->order->id,
        );
    }

    // 設定使用的 Blade 視圖
    public function content(): Content
    {
        return new Content(
            view: 'emails.orders.shipped', // 對應 resources/views/emails/orders/shipped.blade.php
        );
    }
}

```

## 建立郵件視圖(Blade)

建立對應的 blade 視圖

```bash
php artisan make:view emails/orders/shipped

# 生成 app/resources/views/emails/orders/shipped.blade.php
```

編輯 shipped.blade.php，可取得 OrderShipped 的公開屬性 `public Order $order`

```html
<h1>訂單已成功出貨！</h1>
<p>親愛的客戶您好，您的訂單編號為：{{ $order->id }}</p>
<p>總金額：${{ $order->total_price }}</p>

<a href="{{ url('/orders/'.$order->id) }}">查看訂單詳情</a>
```

## 寄出郵件

可以在 Service 或 Controller 使用 Facade `Mail` 寄送電子郵件

```php
public function ship(Order $order)
{
    Mail::to('user@example.com')->send(new OrderShipped($order));
}
```

## 加入附件

若需要在郵件加入附件，可在 Mailable 類別中的 `attachments()` 設定。

```php
public function attachments(): array
{
    return [
        Attachment::fromPath('/path/to/file.pdf'),
    ];
}
```

## 使用 Queue 寄送電子郵件

若要將寄送加入 Queue 佇列，可以讓 Mailable implements `ShouldQueue`

```php
// implements ShouldQueue
class OrderShipped extends Mailable implements ShouldQueue 
{
    // ... Laravel 會自動將郵件放入 Queue 處理
}
```

## 預覽郵件

若要用瀏覽器預覽，可以在 Route 直接回傳 Mailable 實例

```php
Route::get('/mailable/preview', function () {
    $order = App\Models\Order::first();
    return new App\Mail\OrderShipped($order);
});
```

## 使用自訂的郵件內容來源

若不想使用 views 目錄對應的檔案，可以透過 Facade `Blade` 的 render() 方法，在 content 自行渲染郵件的內容並寄出。

```php
public function content(): Content
{
    $body = Blade::render(
        "您的訂單成立於 {{ $now }}, 訂單編號為 {{ $orderId }}",
        [
            'now' => now()->format('Y-m-d H:i:s'),
            'orderId' => $this->order->id
        ]
    );

    return new Content(
        htmlString: $body
    );
}
```

{{< alert type="info" >}}
此方法可以搭配自行在資料庫建立的郵件樣板資料表，讓管理者可以在後台自行修改郵件視圖內容。
{{< /alert >}}
