記錄 LMDE 7 安裝完成後的一些設定與踩雷解法。

## LMDE 7 安裝 kate 或其他應用程式，無法顯示圖示

LMDE 7 使用 apt 安裝 kate 後，發生圖示無法顯示的問題，此時可安裝 `qt6-svg-plugins` 解決。

```bash
# 安裝
sudo apt install qt6-svg-plugins
```

## 使用 fcitx5 取代 fcitx

安裝完 LMDE 7 ，在系統設定的輸入法若要用中文，會要你安裝 fcitx，但是因為預設為 fcitx 4，會導致部份應用程式無法輸入中文，例如 kate。因此建議改安裝 fcitx5。

可參考[Ubuntu繁體中文輸入法？改用Fcitx5來輸入中文吧！](https://ivonblog.com/posts/ubuntu-fcitx5/)

```bash
# 安裝必要項目
sudo apt install fcitx5 fcitx5-configtool fcitx5-chinese-addons fcitx5-fortend-all fcitx5-chewing
```

接著開啟 `Fcitx 5 設定`，將注音從可用輸入法加入到目前輸入法，登出登入或重新開機後，即可使用新酷音輸入中文。

## 將家目錄內容改為英文

安裝完中文版的桌面發行版後，通常家目錄底下的常用目錄也會用中文命名，此時可用 xdg-user-dir-update 指令來調整為英文。

```bash
# 開啟 Terminal
export LANG=C.UTF-8

xdg-user-dirs-gtk-update
```

此時會跳出視窗詢問是否更新資料夾名稱，點擊更新。  
重新登出登入後，會詢問是否保留原本的名稱，請選擇保留(維持英文名稱)。

若發生檔案管理器(Nemo)的左側書籤路徑找不到，則可以先把書籤檔移除

```bash
# 刪除書籤檔
nano ~/.config/gtk-3.0/bookmarks
# 重啟 nemo
nemo -q
```

若要把目錄快捷加回去，可點進該目錄(e.g. ~/Documents) 然後按下 Ctrl + D，建立新的捷徑。

參考資料：

[Ubuntu維持中文界面，將家目錄下的資料夾改成英文 (xdg-user-dirs-update用法)](https://ivonblog.com/posts/linux-xdg-user-dirs/)

## 安裝防毒 ClamAV

```bash
sudo apt install clamav clamav-daemon clamav-freshclam

# (選用)可安裝圖形介面
sudo apt install clamtk

```

首次安裝手動更新病毒碼

```bash
# 1. 暫停更新服務
sudo systemctl stop clamav-freshclam

# 2. 強制執行手動更新（這需要一點時間下載幾百 MB 的病毒碼）
sudo freshclam

# 3. 更新完後，重新啟動並開機自動執行更新服務
sudo systemctl enable --now clamav-freshclam
```

啟動背景服務

```bash
sudo systemctl enable --now clamav-daemon
```

## 安裝 LINE(因windows憑證問題已失效)

可參考以下網址

[【心得】linux 出專欄啦！！！（23）: 自己筆記，新版LINE懶人安裝大法](https://forum.gamer.com.tw/C.php?bsn=60030&snA=677146)

[如何在Linux安裝電腦版Line，使用Bottles管理Wine容器](https://ivonblog.com/posts/linux-bottles-install-line/)

簡化步驟如下

1. 安裝 Bottles 並開啟
2. 右上方 Menu → 偏好設定 → 執行器 → 安裝 kron4ek(選擇新版即可，若之後安裝失敗，再考慮退一個版本)。
3. 建立一個新的 bottle，執行器選擇安裝的 kron4ek。
4. 安裝相依套件 cjkfonts, vcredist2022, d3dcompiler_47。
5. 若字型太小，可調整 DPI：顯示設定區塊 → 進階顯示設定 → Screen Scaling 設定 144。
6. 到 LINE 官往下載 LINE 安裝檔：運行可執行檔 → 選擇下載的安裝檔。

若無法輸入中文：設定 → 相容性區塊 → 環境變數，寫入以下環境變數。

```text
GTK_IM_MODULE=fcitx
QT_IM_MODuLE=fcitx
SDL_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
```

重新啟動。

## 設定 DNS

若在使用 docker 或者 php(curl) 無法正確解析遠端主機時，可能是因為 DNS 解析的問題，可以進入網路設定自行將 IPv4 的 DNS 設定為手動加入以下  

- 8.8.8.8
- 1.1.1.1

## 其他工具

建議安裝的其他工具

### Plank 工具列

Plank 套件自行釘選的工具列。

另外有一個 cairo-dock 一樣是自訂工具列，但是較為複雜。

```bash
sudo apt install plank
```

#### 設定工具列位置

開啟後在 Plank 上的圖示之間（不要點到圖示本身，點擊空白處）按下 Ctrl + 滑鼠右鍵，可選擇「偏好設定」自訂位置。或者使用 command 開啟設定。

```bash
plank --preferences
```

#### 自動啟動

1. 在選單搜尋並開啟 「啟動應用程式」(Startup Applications)。
2. 點擊視窗下方的 「+」 按鈕（或「加入」）。
3. 選擇 「自訂指令」(Custom Command)。
4. 在彈出的視窗中輸入以下資訊
   - 名稱：Plank
   - 指令：plank
   - 註釋：我的桌面 Dock(Plank)
   - 延遲啟動：建議設定 3 ~ 5 秒。
