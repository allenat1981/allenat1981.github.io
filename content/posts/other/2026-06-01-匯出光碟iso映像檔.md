使用 dd 指令將光碟匯出成 iso 檔案。若 dd 無法匯出，則可使用 genisoimage(mkisofs) 工具。

## 使用 dd 匯出 iso

檢查光碟設備代號

```bash
lsblk
# 通常是 sr0(/dev/sr0)
```

```bash
dd if=/dev/sr0 of=/path/to/output.iso bs=4M status=progress
```

- if 輸入路徑
- of 輸出路徑
- bs=4M 設定 4MB 緩衝區
- status=progress 顯示目前進度

## 使用 genisoimage 匯出 iso

mkisofs 在 Debian 是 genisoimage 的別名。

使用 genisoimage 匯出時，需先將光碟掛載(現代 Linux 系統通常會自動掛載)

使用 `df -h` 或 `lsblk` 可找到掛載的目錄。接著使用 genisoimage 指令進行匯出

```bash
genisoimage -R -J -V "DVD_LABEL" -o ~/path/to/output.iso /media/from/mounted
```

-o <檔案>   output  指定輸出的 ISO 檔案路徑。
-R  Rock Ridge  允許 Linux 長檔名、大小寫與權限保留。
-J  Joliet  允許 Windows 長檔名，防止中文亂碼。
-V <名稱> 設定光碟標籤名稱（掛載時顯示的名字。
